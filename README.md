Project-Hygieia

Phase 1 -

Application is a document management system where users can register and upload documents. The application will have the following features:
1) User Registration: Users can register by providing their details such as name, email, and password. The system will validate the input and create a new user account.
2) User can upload documents by providing the document details such as title, description, and the file itself.
3) These files are segregated based on the document category -  Graduation_Document, Government_ID, Corporate_Document, Medical_Document, Other
4) These documents can be viewed on the UI by the category or by All categories.

Phase 2 -

Application will evolve to take up medical documents and will have the following features:
1) Medical document such as Reports, Prescriptions will be saved and categorized.
2) These documents will be processed by AI to have the Prescription date, Doctor's name, Medicines under the prescriptions, Link to buy the prescribed medicines.

Phase 3 -

Application will evolve to be accessible via mobile web app which requires no installation.
